## FAQ

**Q: How do I unlock the next kanji level?**  
A: Complete all required radicals/components and answer correctly 5 times.

**Q: How does the spaced repetition system work?**  
A: PanKan schedules reviews to help you remember kanji efficiently over time.

**Q: Can I use the app offline?**  
A: Yes, but syncing progress requires an internet connection.
